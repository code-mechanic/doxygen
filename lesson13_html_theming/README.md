# Lesson 13

This lesson focuses on changing the look of Doxygen's HTML output without
rewriting the whole generator.

- built-in palette tuning with `HTML_COLORSTYLE_*`
- project branding with `PROJECT_LOGO`
- custom styling with `HTML_EXTRA_STYLESHEET`
- practical CSS selector targets in Doxygen output

The sample is a small fault-journal module written in embedded-style C.
