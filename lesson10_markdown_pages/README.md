# Lesson 10

This lesson shows how to write Doxygen pages as Markdown files instead of
`.dox` files.

- Markdown-based pages in `pages/*.md`
- normal Markdown headings, lists, and fenced code blocks
- Doxygen commands mixed into Markdown with `@page`, `@ref`, `@snippet`, and `@dot`
- example pages and source-linked snippets
- call graphs and source browsing for the C implementation

The sample is a small boot-control module written in embedded-style C.
