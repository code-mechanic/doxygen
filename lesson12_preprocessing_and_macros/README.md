# Lesson 12

This lesson focuses on how Doxygen handles preprocessing and macro-heavy
embedded C.

- `ENABLE_PREPROCESSING`
- `MACRO_EXPANSION`
- `EXPAND_ONLY_PREDEF`
- `PREDEFINED`
- `EXPAND_AS_DEFINED`
- build-flag-driven conditional APIs

The sample is a small diagnostic-log module that uses feature flags and macro
lists in a way that is common in embedded projects.
