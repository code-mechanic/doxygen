# Lesson 9

This lesson shifts away from API comments alone and explores some of the other
Doxygen features that become useful once a codebase grows.

- `@page`, `@section`, and `@tableofcontents`
- Markdown-driven main pages
- cross-references with `@ref`
- example pages with `@example`
- snippet extraction with `@snippet`
- diagrams with `@dot`
- source browsing, inline source, and call graphs

The sample is a small firmware-update module written in embedded-style C.
