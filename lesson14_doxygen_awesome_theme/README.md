# Lesson 14

This lesson uses the upstream `doxygen-awesome-css` theme instead of a
handwritten stylesheet.

- vendoring a third-party theme asset
- enabling the theme through `HTML_EXTRA_STYLESHEET`
- keeping attribution and license text with the lesson
- observing how the theme styles notes, warnings, todos, and deprecated APIs

The sample is a small watchdog-recovery module written in embedded-style C.
